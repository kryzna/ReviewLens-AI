# Spec: Above-and-Beyond UX Features

## 1. Mandate

Add three AI-powered UX upgrades to the session chat panel: (1) quick-start prompts shown in the empty state, (2) contextual follow-up chips generated by the LLM after each assistant response, and (3) a proactive insight brief auto-generated on session load and displayed as a collapsible card above the chat.

## 2. Tech stack

- Next.js 15 App Router, TypeScript 5
- Anthropic SDK `@anthropic-ai/sdk` (already installed), model `claude-sonnet-4-6`
- `pg` (node-postgres) — no schema changes required
- React state + `useEffect` for client-side loading

## 3. Data models

### InsightBrief (client type, returned by `/api/sessions/[id]/insight`)
```typescript
interface InsightTheme { title: string; description: string; quote: string; }
interface InsightBrief {
  sentiment: 'positive' | 'mostly positive' | 'mixed' | 'mostly negative' | 'negative';
  score: number;   // weighted avg star rating 1.0–5.0
  summary: string; // one-sentence overview
  themes: InsightTheme[]; // exactly 3
}
```

### FollowupsResponse (returned by `POST /api/sessions/[id]/followups`)
```typescript
interface FollowupsResponse { suggestions: string[]; } // 2–3 items
```

## 4. Non-goals

- No DB caching of insight briefs or follow-up chips (fetched fresh per session load)
- No pagination of themes
- No user ability to regenerate insight brief
- No streaming for insight or followups endpoints (short responses, non-streaming is fine)

## 5. Boundary conditions

- Never commit secrets or `.env` files
- If LLM returns malformed JSON, catch and suppress — component shows nothing gracefully
- Insight brief caps reviews at 200 to bound token cost
- Follow-up chips shown only for the most recent assistant message
- Follow-up chips dismissed when clicked (sent as next user message)

## 6. Escalation protocol

Encounter missing dependency, conflicting schema, ambiguous requirement, or contradiction between this spec and existing codebase: **stop**. Describe blocker one paragraph, propose 2–3 options with tradeoffs, ask clarification.
